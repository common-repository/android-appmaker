Mit diesem Plugin kannst du für deine Seite eine Android-App erstellen. iOS Apps können auf Anfrage erstellt werden. Nach wenigen Klicks sind deine Inhalte auch in der App verfügbar!

== Description ==

*Dieses Plugin ist die Schnittstelle zwischen deiner Seite und der App.*

Nachdem du dieses Plugin installiert hast, kannst du dir eine App bei [Appmaker](https://appmaker.merq.org) erstellen.
Diese kann <span style=\"text-decoration: underline;\">kostenlos</span> oder im Pro-Tarif mit ein paar Extras genutzt werden.

<span style=\"text-decoration: underline;\">Die wichtigsten Funktionen im Überblick:</span>

*   App enthält immer die aktuellsten Artikel
*   Aussehen der Applikation kann angepasst werden
*   Push-Nachrichten und eigene Werbung im Pro-Tarif

== Installation ==

1. Installiere das Plugin im Admin-Bereich oder extrahiere die ZIP-Datei, lade sie in den Ordner wp-content/plugins/ hoch und aktiviere das Plugin im Admin-Bereich.
2. Gehe im Admin-Bereich auf Einstellungen -> Android Appmaker und befolge die Anweisungen.

== Frequently Asked Questions ==

= Ist das Erstellen einer App wirklich kostenlos? =

Ja, Appmaker ist kostenlos. Die App kann kostenpflichtig erweitert werden (Pro-Tarif). Weitere Informationen dazu unter https://appmaker.merq.org/tarife

= Wo kann ich diverse Einstellungen ändern? =

Sie können im Administrationsbereich und unter appmaker.merq.org Einstellungen ändern.
